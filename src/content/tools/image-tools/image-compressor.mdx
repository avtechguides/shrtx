# Image Compressor

Compress images to reduce file size without losing visual quality.

<ImageCompressor client:only="react"/>

## How it works

- Step 1: Upload an image.
- Step 2: Tool compresses it instantly in the browser.
- Step 3: Download or copy the optimized image.

## Features

- Instant compression in browser.
- Supports multiple formats (PNG, JPEG, WebP).
- Works offline after first load.

## Use cases

- Optimize images for website speed.
- Reduce storage usage without losing quality.
- Prepare images for email campaigns.

## Tips

- Compare before and after quality.
- Use batch compression for multiple images.
- Avoid excessive compression to retain clarity.
