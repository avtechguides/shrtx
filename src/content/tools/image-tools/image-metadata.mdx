# Image Metadata Viewer

Inspect your images for detailed metadata including EXIF, dimensions, and camera info.

<ImageMetadata client:only="react"/>

## How it works

- Step 1: Upload an image.
- Step 2: Tool displays metadata instantly.
- Step 3: Copy or review data for analysis.

## Features

- View EXIF, dimensions, file size, and camera model.
- Works offline after first load.
- Instant browser processing.

## Use cases

- Check camera settings for photography.
- Validate image information for projects.
- Analyze uploaded images for technical details.

## Tips

- Use with large images for full EXIF details.
- Copy metadata for cataloging.
- Check privacy-sensitive info before sharing images.
