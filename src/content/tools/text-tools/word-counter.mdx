# Word Counter

Quickly count words in your text to track length and readability for writing or editing.

<WordCounter client:only="react" />

## How it works

- Step 1: Paste or type your text.
- Step 2: Tool automatically calculates word count.
- Step 3: Review counts and copy results if needed.

## Features

- Real-time word counting.
- Character and line count included.
- Works offline after first load.

## Use cases

- Check article length for SEO.
- Monitor essay or report word limits.
- Edit content with precise word control.

## Tips

- Use keyboard shortcuts for faster input.
- Remove unnecessary whitespace for accurate counts.
- Paste large texts in chunks if browser slows.
