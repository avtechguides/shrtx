# URL Shortener

Quickly shorten long URLs for easy sharing across social media, email, or messages.

<URLShortener client:only="react" />

## How it works

- Step 1: Paste the long URL in the input field.
- Step 2: Click "Shorten" to generate a short URL.
- Step 3: Copy the output and share instantly.

## Features

- Instant processing in the browser (no server upload).
- Works offline after first load (PWA-ready).
- Keyboard shortcuts and accessible labels included.

## Use cases

- Share long URLs easily on social media.
- Simplify links in emails or documents.
- Track clicks on short URLs for analytics.

## Tips

- Use descriptive aliases for better recognition.
- Test the short URL before sharing widely.
- Bookmark frequently used URLs for quick shortening.
