# PDF to Text / OCR

Convert PDFs into editable text quickly, even scanned documents, directly in the browser.

<PdfToText client:only="react"/>

## How it works

- Step 1: Upload PDF file.
- Step 2: Tool extracts text or applies OCR.
- Step 3: Copy or download extracted text.

## Features

- Extracts text from normal and scanned PDFs.
- Works offline after first load.
- Instant processing in browser.

## Use cases

- Extract content from reports, forms, and eBooks.
- Copy text for notes or research.
- Process scanned documents efficiently.

## Tips

- Check extracted text for formatting issues.
- Use for searchable PDFs.
- Combine with PDF Compressor if file size is large.
