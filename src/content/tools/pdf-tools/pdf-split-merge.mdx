# PDF Split / Merge

Split or merge PDF files instantly in the browser without uploading to a server.

<PdfSplitMerge client:only="react"/>

## How it works

- Step 1: Upload PDF(s).
- Step 2: Choose split ranges or merge order.
- Step 3: Download processed PDF files.

## Features

- Split large PDFs into smaller files.
- Merge multiple PDFs in desired order.
- Works offline after first load.

## Use cases

- Extract specific pages for sharing.
- Combine multiple reports into a single file.
- Reduce file size by splitting large PDFs.

## Tips

- Preview selected pages before splitting.
- Ensure correct merge order.
- Use along with PDF Compressor for optimal size.
