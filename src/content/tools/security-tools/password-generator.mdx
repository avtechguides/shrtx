# Password Generator & Strength Checker

Generate strong, secure passwords and verify their strength to protect accounts and sensitive data.

## How it works

- Step 1: Select password length and character sets.
- Step 2: Tool generates a password instantly.
- Step 3: Check strength and copy for use.

## Features

- Instant secure password generation.
- Strength assessment for existing passwords.
- Works offline after first load.

- Create strong passwords for online accounts.
- Test existing passwords for vulnerabilities.
- Ensure compliance with security policies.

## Tips

- Use longer passwords (12+ characters) for better security.
- Include letters, numbers, and symbols.
- Avoid reusing passwords across sites.
