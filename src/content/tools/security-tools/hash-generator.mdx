# Base64 Encode / Decode

Quickly encode text to Base64 or decode Base64 to its original form for programming or web use.

<Base64Tool client:only="react"/>

## How it works

- Step 1: Paste text to encode or decode.
- Step 2: Select Encode or Decode mode.
- Step 3: Copy output for use.

## Features

- Encode/decode instantly in-browser.
- Works offline after first load.
- Simple interface for fast conversions.

## Use cases

- Convert text for web applications.
- Encode data for email or programming purposes.
- Decode Base64 content from other sources.

## Tips

- Remember Base64 is not encryption.
- Useful for embedding images or text in code.
- Use with caution for sensitive info.
